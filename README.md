# Temperature Converter

A web-based temperature converter that converts between Celsius, Fahrenheit, and Kelvin in real-time. Simply enter the temperature in one unit, and the corresponding values in the other two units will be automatically calculated.

## Features:
- Converts between Celsius, Fahrenheit, and Kelvin
- Responsive design with intuitive UI
- Real-time conversion as you type

## Technologies Used:
- HTML
- CSS
- JavaScript

## Usage:
1. Clone the repository.
2. Open the `index.html` file in your browser.
3. Enter a temperature value in any of the input fields (Celsius, Fahrenheit, or Kelvin), and the values in the other fields will update automatically.

## Setup:
No external dependencies are required. Just download or clone the repo and open the `index.html` file in any modern web browser.
